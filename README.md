# Building AI Applications with Foundation Models

#### July 11, 2026 11:00 AM CST - 2:00 PM CST 

Welcome to the source code repository for the **Building AI Applications with Foundation Models** workshop.

Apple's Foundation Models framework makes it possible to bring powerful, on device AI experiences directly into your iOS applications. In this hands on workshop, you'll learn how to integrate Foundation Models into SwiftUI applications using practical examples that you can immediately apply to your own projects.

We'll begin by exploring how to generate structured responses using `@Generable`, allowing the model to return strongly typed data instead of plain text. From there, you'll learn how to extend the model with custom tools, giving your AI assistant access to your application's capabilities.

Throughout the workshop, you'll build a complete SwiftUI application that demonstrates how to send prompts, stream responses, display AI generated content, and create a polished user experience. Finally, you'll learn how to persist conversations and generated content using SwiftData so users can return to previous sessions.

## What You'll Learn

* Understand the Foundation Models framework and when to use it
* Generate strongly typed responses using `@Generable`
* Build custom tools that Foundation Models can invoke
* Stream AI responses in real time
* Integrate Foundation Models into SwiftUI applications
* Design responsive AI powered user interfaces
* Store AI generated content using SwiftData
* Apply best practices for building production ready AI applications

## Prerequisites

To get the most from this workshop, you should have:

* A working knowledge of Swift and SwiftUI
* Xcode 26 or later
* macOS Tahoe or later
* A supported Apple device capable of running Foundation Models (recommended)
